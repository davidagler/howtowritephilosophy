# Arguing Assignment

In this assignment you will be given an argument and then asked to put it into argument standard form.

## Step-by-step instructions

1. Download the `arguing` template file from the templates folder. You can copy and paste the raw file here: [arguing template](https://github.com/davidagler/howtowritephilosophy/blob/main/templates/arguing/arguing_template.md?plain=1)
2. In each section, there is an argument. Below each argument, rewrite it in standard form. *Do not change of the wording of the argument*.
3. Save the file and submit it.
