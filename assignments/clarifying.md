# Clarifying Assignment

In this assignment you are asked to select a passage of text from a text, place it in a block quote, and then clarify it.

## Step-by-step instructions

1. Download the `clarifying` template file from the templates folder. You can copy and paste the raw file here: [clarifying template](https://github.com/davidagler/howtowritephilosophy/blob/main/templates/clarifying/clarifying_template.md?plain=1)
2. Replace the dummy text in the block quotation with a quotation that you wish to clarify.
3. Supply the quote with an in-text citation using the format `(Author Date: Page)`.
4. Using an unordered list, add reference citations in the *References* section.
    - For books, use the following format: `Author. Date. *Book Title*. Location of Publication: Place of Publication.` Here is an example: Agler, David W. 2026. *An Introduction to Metaphysics: Identity, Free Will, and Time.* University Park: Penn State University Multimedia and Print Center.
    - For articles, use the following format: `Last Name, First Name. Date of Publication. Article Title. *Journal Title*. Volume Number(Issue Number): Page Number(s).`
5. Clarify the block quotation in two separate sections. In the first section, try to explain what the passage means by either (1) making it more precise or (2) making it more understandable. In the second section, create an example that helps to illustrate the meaning of the block quotation.
6. Save the file and submit it.
